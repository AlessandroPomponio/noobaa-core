Domain Name System: The Internet's system for converting alphabetic names into numeric addresses.

Before you apply changes, add the following entry to your DNS:

IP: Name:
Changing DNS name procedure:
  1. Add the new name in your DNS to have both old and new names avaliable
  2. Apply the changes in the forms above
  3. Once all nodes are updated with the new name, the old name can be removed from the DNS

In any case, reinstalling a node (from the install node wizard) will update the server address used by that node.
note: make sure that the server IP was configured Static and not DHCP, otherwise if the IP will suddenly change, the nodes will lose connectivity.
