Notes about using a DNS name (recommended):

  • If you want to change the name - it is recommended to first add the new name in your DNS to have both old and new names available, then apply the change here in the system, and only once all nodes are updated with the new name to remove the old name from the DNS.
  • If you are unable to use the suggested order of actions - nodes that disconnected from the server will not be able to connect using the old address to receive the update on the new name, and will need to be reinstalled.

Notes about using an IP address:

  • Make sure that the server IP was configured Static and not DHCP, otherwise if the IP will suddenly change the nodes will loose connectivity.
  • If you want to change this IP - you will have to reinstall the node agents manually as there is no way for the server to communicate with them once they disconnected and do not yet
